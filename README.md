
# Kit

`kit` 은 Rust 기반의 API 서버 템플릿입니다. 빠르게 확장 가능한 백엔드 애플리케이션을 구축하기 위한 기본 구조를 제공합니다.

### Stacks

- rust 2024 edition
- ntex.rs
- postgresql
- sqlx
- redis
- nats
- docker

### Getting Started

먼저 러스트를 설치합니다. 현재 `Kit` 은 최신 버전인 <u>2024 edition<u> 을 사용하고 있습니다.

```Shell
rustup install 2024
```

프로젝트를 컴파일 합니다.

```Shell
$ cargo build
```

프로젝트를 시작합니다.

```Shell
cargo run main.rs
```
